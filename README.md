# tiny
A collection of single header libraries focused on game development

## Features
- No dependencies other than the C Standard Library

## Libraries

| :--: | :-----: | :-: | ----------- |

| [tiny_ds.h](https://github.com/nosbod18/tiny/blob/main/tiny_ds.h)     | 0.2 | 171 | Generic data structures                                    |
| [tiny_gfx.h](https://github.com/nosbod18/tiny/blob/main/tiny_gfx.h)   | 0.2 | 372 | 3D graphics                                                |
| [tiny_math.h](https://github.com/nosbod18/tiny/blob/main/tiny_math.h) | 0.4 | 491 | Vector, matrix, affine transformation, and projection math |
