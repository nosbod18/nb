# nb

`nb` is a few cross-platform single header libraries written in C99. Collectively, the libraries are indented to provide a
jumping-off point for your own game or engine development.

Each library is licenced under the public domain.

|                                 Library                                | Version |  LoC  |               Description              |
| :--------------------------------------------------------------------- | :-----: | :---: | -------------------------------------- |
| [nb_ds.h](https://github.com/nosbod18/nb/blob/main/nb_ds.h)            |  0.10   |  212  | Generic data structures                |
| [nb_ecs.h](https://github.com/nosbod18/nb/blob/main/nb_ecs.h)          |  0.04   |  471  | Minimal Entity Component System        |
| [nb_gl.h](https://github.com/nosbod18/nb/blob/main/nb_gl.h)            |  0.03   |  334  | OpenGL abstraction                     |
| [nb_math.h](https://github.com/nosbod18/nb/blob/main/nb_math.h)        |  0.06   |  330  | 2D and 3D vector and matrix math       |
| [nb_platform.h](https://github.com/nosbod18/nb/blob/main/nb_platform.h)|  0.06   |  659  | Platform window and input abstraction  |

**Total LoC:** 1674

## Features
- Written in pure C99
- No external dependencies
- No inter-dependencies between libraries
- Simple APIs
