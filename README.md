# nb

`nb` is a collection of single header libraries written in C99.
Collectively, the libraries are intended to provide a jumping-off point for your own game or engine development.

|                          Library                          | Version |  LoC   |                    Description                    |
| ----------------------------------------------------------| :-----: | :----: | ------------------------------------------------- |
| [app.h](https://github.com/nosbod18/nb/blob/main/app.h)   |  0.06   |   620  | Platform window, input and main loop abstraction  |
| [ds.h](https://github.com/nosbod18/nb/blob/main/ds.h)     |  0.10   |   182  | Generic data structures                           |
| [ecs.h](https://github.com/nosbod18/nb/blob/main/ecs.h)   |  0.04   |   454  | Minimal Entity Component System                   |
| [gl.h](https://github.com/nosbod18/nb/blob/main/gl.h)     |  0.03   |   320  | OpenGL abstraction                                |
| [math.h](https://github.com/nosbod18/nb/blob/main/math.h) |  0.06   |   304  | 2D and 3D vector and matrix math                  |

## Features
- Written in pure C99
- Public domain license
- No external dependencies
- No inter-dependencies between libraries
- Simple APIs
