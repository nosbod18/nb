# nb

`nb` is a collection of single header libraries written in C99.
Collectively, the libraries are intended to provide a jumping-off point for your own game or engine development.

|                          Library                          |  LoC   |                    Description                    |
| ----------------------------------------------------------| :----: | ------------------------------------------------- |
| [app.h](https://github.com/nosbod18/nb/blob/main/app.h)   |   628  | Platform window, input and main loop abstraction  |
| [ecs.h](https://github.com/nosbod18/nb/blob/main/ecs.h)   |   439  | Minimal Entity Component System                   |
| [gl.h](https://github.com/nosbod18/nb/blob/main/gl.h)     |   305  | OpenGL abstraction                                |
| [log.h](https://github.com/nosbod18/nb/blob/main/log.h)   |    53  | Simple logger                                     |
| [map.h](https://github.com/nosbod18/nb/blob/main/map.h)   |    98  | Generic Hashmap                                   |
| [math.h](https://github.com/nosbod18/nb/blob/main/math.h) |   147  | Vector and matrix math                            |
| [vec.h](https://github.com/nosbod18/nb/blob/main/vec.h)   |    47  | Generic dynamic array                             |

## Features
- Written in pure C99
- Public domain license
- No external dependencies
- No inter-dependencies between libraries
- Simple APIs
